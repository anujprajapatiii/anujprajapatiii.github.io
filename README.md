# Anuj Prajapati — Personal Portfolio

Personal portfolio site, built with Astro and maintained with Claude Code.

- Live site: https://anujprajapatiii.github.io
- Stack: Astro v5 (static), Tailwind CSS v4, shadcn/ui, React for interactive islands
- Deployment: GitHub Pages via GitHub Actions on pushes to `main`

## Develop

```bash
pnpm install
pnpm dev      # http://localhost:4321
pnpm build    # production build to dist/
```
