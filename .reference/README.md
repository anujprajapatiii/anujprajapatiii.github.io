# Reference

User-provided briefs, screenshots, design references, and exported content
from the previous portfolio live here. Agents may read this folder for
context but should never publish from it directly.

Suggested structure:

- `brief.md` — the one-paragraph site brief and scope notes
- `screenshots/` — current portfolio and reference sites
- `content/` — exported case-study text and images awaiting migration
- `tokens/` — color and type explorations
